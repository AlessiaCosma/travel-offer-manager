# Travel Offers Management Application

This software application facilitates travel planning in tourism through multi-criteria filtering of transport and accommodation offers. Users can enter criteria such as departure city, destination, price range, and additional preferences, and the application generates alternative travel options and accommodation suggestions.

## Features

- Multi-criteria filtering of transport and accommodation offers.
- Comparison of different means of transport.
- Data visualization with Matplotlib.
- Intuitive graphical user interface built with Tkinter.
- Uses XML files and external APIs for data handling.

## Technologies

- Python
- Tkinter (graphical interface)
- Matplotlib (charts and plots)
- External APIs
- XML files for data management

## Usage

> Currently, the application is under development. `main.py` will serve as the main entry point. A `.env` file is required for application configuration.

The application allows users to input travel criteria and generates corresponding transport and accommodation alternatives based on preferences.

## Limitations

- Data accuracy depends on external sources and available files.
- The application is still in development, and some functionalities are not yet completed.

## Project Goal

The project aims to simplify travel planning by providing users with a quick and visual way to compare different transport and accommodation options, with an intuitive and clear workflow.
